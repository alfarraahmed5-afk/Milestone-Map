# Journey Map: Copenhagen → Lisbon

A freelance illustrator briefing package for a mobile app overworld map feature.

This map represents a virtual group journey across 18 checkpoints from Copenhagen to Lisbon, rendered in a paper/vintage illustration style for integration into a mobile app.

## Documents

| Document | Description |
|---|---|
| [docs/SOW.md](docs/SOW.md) | Statement of Work — scope, deliverables, pricing, and timeline |
| [docs/MVP.md](docs/MVP.md) | MVP Definition — first 3 checkpoints, deliverables checklist, success metrics |
| [docs/BUILD_SPEC.md](docs/BUILD_SPEC.md) | Build Specification Sheet — canvas, layout algorithm, checkpoint states, geographic elements, visual style |

## Journey Route

18 checkpoints (bottom → top on map):

1. Copenhagen
2. Odense
3. Hamburg
4. Hannover
5. Cologne (Keulen)
6. Brussels
7. Paris
8. Orléans
9. Bordeaux
10. San Sebastián
11. Burgos
12. Valladolid
13. Madrid
14. Ávila
15. Salamanca
16. Vilar Formoso
17. Coimbra
18. Lisbon
